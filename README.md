​🎵 Code Harmony: Visualizing Logic as Music
​Code Harmony는 딱딱한 소스 코드를 감각적인 펜타토닉 선율과 비트로 변환해주는 오디오 비주얼라이저 프로젝트입니다. 코드의 문자를 음표로, 키워드를 리듬으로, 그리고 코드의 깊이(Depth)를 베이스의 웅장함으로 바꾸어 코딩의 과정을 예술로 승화시킵니다.
​✨ Key Features
​Pentatonic Quantization: 어떤 코드를 입력해도 불협화음 없이 조화로운 C Major Pentatonic 음계로 변환합니다.
​Logical Rhythm: * if, for, while 등 주요 키워드는 경쾌한 하이햇(Hi-hat) 소리를 생성합니다.
​중괄호({, })는 심장을 울리는 킥 드럼(Kick) 비트가 됩니다.
​Deep Bass Drop: C++ 엔진으로 분석된 코드의 **중첩 깊이(Depth)**가 3단계 이상이 되면, 오실레이터의 주파수가 절반으로 낮아지며 묵직한 베이스 사운드를 선사합니다.
​Neon Visualization: 연주되는 문자들이 캔버스 위에서 네온 빛으로 춤추며 시각적인 즐거움을 더합니다.
​🛠 Tech Stack
​Frontend: HTML5, CSS3, JavaScript (Web Audio API, Canvas API)
​Logic Engine: C++ (Code Depth Analysis Logic)
​Collaboration: Gemini (Google AI)
​🤝 AI Collaboration
​이 프로젝트는 Google의 AI 모델인 Gemini와의 협업을 통해 제작되었습니다.
​Gemini는 Web Audio API의 오디오 합성 로직 설계, 펜타토닉 스케일 맵핑 알고리즘 구현, 그리고 소리 최적화 과정에서 기술적인 가이드와 코드를 제공하며 페어 프로그래밍 파트너로서 함께했습니다.
​🚀 How to Use
​index.html을 브라우저에서 엽니다.
​음악으로 듣고 싶은 C++ 혹은 C# 코드를 입력창에 붙여넣습니다.
​[Convert & Play] 버튼을 누르고 코드가 만들어내는 선율을 감상하세요.
​소리가 너무 날카롭다면 Speed를 낮추거나 Sine 파형을 선택하세요.
​Note: 이 프로젝트는 코드의 실행 가능 여부보다, 코드에 담긴 문자의 배열과 구조적 미학에 집중합니다.
