About NEON_SYNC Project

이 프로젝트는 복잡한 프로그래밍 코드를 시각과 청각 데이터로 변환하여, 코드를 '읽는' 것을 넘어 '느낄' 수 있게 만드는 것을 목표로 합니다.

현재 C/C++ 기반의 파싱 엔진과 Web Audio API를 연동하여 다이나믹한 공감각적 경험을 제공하고 있습니다.


Tech Stack

  • Frontend: HTML5 Canvas, CSS3 (Cyberpunk Theme), Vanilla JavaScript
  • Audio: Web Audio API (Oscillator, BiquadFilter)
  • Backend/Engine: C++ (WebAssembly Ready)
