About NEON_SYNC

복잡한 프로그래밍 코드를 시각과 청각 데이터로 변환하여, 코드를 '읽는' 것을 넘어 '느낄' 수 있게 만드는 프로젝트입니다.


Tech Stack

  • Frontend: HTML5 Canvas, CSS3 (Glassmorphism), Vanilla JS
  • Audio: Web Audio API (Oscillator, Quantization)
  • Backend/Engine: C++ (WebAssembly Ready)
